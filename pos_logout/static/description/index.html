Lock POS Screen

Users/Cashiers can unlock POS screen by scanning their barcode or using security PIN

Key features:
  • lock POS screen if necessary
  • unlock screen using barcode
  • unlock screen using security PIN

How It Works

Go to Settings → Users menu and open a user form. Be sure that Barcode and/or Security PIN are set in the Point of Sale tab.


Go to Point of Sale → Dashboard and open POS session. Click on user name at the top of POS screen. Then click the logout icon to lock the screen.



To unlock it, you have to make one of the following:

  • scan user barcode when screen locked
  • tap on the screen and scan user barcode when user selector popup opened
  • select a user and input security PIN



If the barcode or pin is correct, POS is unlocked succesfully.


Otherwise, you will face the warning.
