POS Print Method

Choose print method for order printing in POS

The module allows to choose print method for POS orders:

  • Order. Print entire order on one receipt (default odoo method)
  • Order Line. Print each order line on separate receipts

How It Works

Go to Point of Sale → Configuration → Order Printers menu and open order printer form

In the "Print Method" field select one of 2 methods:

  • Order. Print entire order on one receipt (default odoo method)
  • Order Line. Print each order line on separate receipts

Open POS interface and add products. Click on the "Order" button.

You will get either one receipt for entire order or several receipts per each order line depending on the method you have selected for your order printer(s).
