Customizable POS Receipt

Customize POS receipt to your taste

The module allows to customize POS receipt by adding custom templates. You can create as many templates as you wish and implement them by easy way.

How It Works

Go to Point of Sale → Configuration → Point of Sale.
To apply PosBox Receipt Template:

  • Activate Receipt Printer
  • Check Custom PosBox Receipt box
  • Select a custom template or add new one at Custom PosBox Receipt Template

To apply POS Ticket Template:

  • Check Custom box
  • Here also select a custom template or add new one in Custom Template field
