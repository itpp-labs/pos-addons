Disable options in POS (restaurant extension)

Control access to POS Restaurant options

How It Works

Go to Settings → Users menu and open user form. You can see new options Allow change Qty for kitchen orders and/or Allow remove kitchen order line in the Point of Sale tab. Let's uncheck them for example.


Open POS session. Add products allowed to be sent to kitchen and click Order button. As a result, Qty button is disabled.


Then if you set orderline quantity to 0, Backspace button is disabled too
