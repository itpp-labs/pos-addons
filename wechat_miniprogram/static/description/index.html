WeChat mini-program API

Basic tools to integrate Odoo and WeChat mini-program

Follow instructions of WeChat API module.
If you have several databases, you need to check that all requests are sent to the desired database. The user authentication request from the Mini-program does not contain session cookies. So, if Odoo cannot determine which database to use, it will return a 404 error (Page not found). In order for the requests to send to the desired database, you need to configure dbfilter.
