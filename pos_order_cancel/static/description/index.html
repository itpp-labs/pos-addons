POS Order Cancel

Specify the reasons for cancel separate products or entire order after printing on order printer

Key features:
  • cancel separate products or entire order after printing on POS order printer and specify cancellation reason
  • set predefined cancellation reasons
  • see the reasons on order receipt
  • track all cancelled orders and reasons in backend

How It Works

Open POS session and add products to order. Click on the 'Order' button to print it on order printer.

Once you have sent order to printer, you are able to cancel entire order or separate products and specify the cancellation reason.

Track all cancelled orders

You can see all cancelled orders and the corresponding reasons in your backend. To see all cancelled orders, go to Point of Sale → Orders menu. They are marked as 'Cancelled'. To see the order cancellation reason, open order form and go to the 'Notes' tab. So it will allow you to track all such orders at any time.

Predefined cancellation reasons

Create predefined cancellation reasons at the Point of Sale → Configuration → POS Product Cancellation Reason menu.
