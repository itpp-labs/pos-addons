Keyboard support in Point Of Sale
================================================================
Module allows to use usual keyboard (not virtual one) in Point of Sale.

Usage:
------

=========== ===================== =================
Type        Numpad                Extra keys  
=========== ===================== =================
mode qty    ``/``                 ``q``
----------- --------------------- -----------------
mode disk   ``-``                 ``d``
----------- --------------------- -----------------
mode price  ``*``                 ``p``
----------- --------------------- -----------------
+/-         ``+``                 ``s``
=========== ===================== =================

Tested on Odoo 8.0 612b44648ee79c5f8bb10e16efdee3cc81abaa08
