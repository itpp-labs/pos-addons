Partner deselection

POS Partner deselection in opened POS on expiry the predefined time interval after the customer was set.

It's a good solution to eliminate human error, avoid fraud in case of customers are changing their choice.
This module automatically allows deselecting customers after some amount of time.

How It Works

  • User with the "POS manager" level of access set up the deselection interval
  • User with the regular level of access (or higher) select the Customer for the purchasing
  • Customer has been deselected after 5 minutes (as example)
  • User can proceed with further work
