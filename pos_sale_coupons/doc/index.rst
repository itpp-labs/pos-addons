=============
 POS Coupons
=============

Installation
============

* `Install <https://odoo-development.readthedocs.io/en/latest/odoo/usage/install-module.html>`__ this module in a usual way


Usage
=====

* Open ``Sales >> Products >> Coupon Program``

  * Click ``[Create]``
  * Specify the fields:

    * Based on Products
    * Quantity
    * Reward
    * Reward Line Product
    * POS Product

* Go to ``Sales >> Configuration >> Coupon Program``

  * Open POS session
  * Click ``[Sell Coupon]``
