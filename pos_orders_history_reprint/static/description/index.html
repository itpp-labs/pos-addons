Reprint POS Orders Receipt

Reprint receipts of paid POS orders

Key features:
  • All features from POS Orders History
  • Reprint receipts of paid POS orders

How It Works

  • Open POS session and click History
  • Click Reprint for an order from the list

As a result, you can see the printed copy of the order receipt


Configuration

Go to Point of Sale → Configuration → Point of Sale and open POS configuration form. Check the Reprint Orders box to activate the order receipt reprinting option.
