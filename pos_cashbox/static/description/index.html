Open CashBox from Backend

New option to open cashbox from POS session form

Usually, Odoo allows to open cashbox (cashdrawer) from POS Payment screen only. In some cases it may be required to open cashbox when Opening or Closing control for a POS session. So the module gives the possibility to do that by using new button on POS session form.

How It Works

Go to Point of Sale → Configuration → Point of Sale menu and open POS configuration form. Activate Cashdrawer and Cash Control. Be sure that cashdrawer is connected.

  • Go to Point of Sale → Orders → Sessions menu
  • Open POS session form
  • See new Open CashBox button
NOTE: the button is available in Opening Control and Closing Control status

Once you click the button, cashbox is opened
