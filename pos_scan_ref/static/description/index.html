FIX searching product by ref in POS

By default, if you put ean13 barcode in "Internal Reference" field of product form, then when you scan item in pos Odoo cannot find product.

This module fix this behaviour. It can be usefull if you use "Internal Reference" field for barcodes, because you have products with a different type of barcodes (e.g. EAN8, JPC, UPC, GTIN, etc.) and you don't want to spend time to put ean13 codes separately at "EAN13 Barcode" field.
